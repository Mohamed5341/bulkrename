# Bulk File Renamer
A python script that renames multiple files using a specific format.

---
# Features
- [ ] Basic rename files (just add name and numbers).
- [ ] Preview before execution.
- [ ] Rollback changes.
- [ ] Modify specific files with certain filters.
- [ ] Use excel file for taking user names map.
- [ ] Rename based on dates of creation or modification.
- [ ] Create GUI.
